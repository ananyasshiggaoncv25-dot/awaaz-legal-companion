# AWAAZ: Anonymous Legal Companion ⚖️

Built for the **Build for Good Hackathon** under the **Voice, Safety & Social Access** theme.

AWAAZ is a frictionless, 100% anonymous legal assistant designed for Indian citizens. It helps users understand their rights and draft formal legal applications for daily grievances (Cyber harassment, RTI, Labor disputes, Pensions) without requiring a lawyer or any user registration.

## 🚀 Features
- **Zero Friction:** No Login, No Sign-up, No Auth.
- **Dynamic Wizard:** Custom paths for Youth, Working Adults, and Seniors.
- **Jargon Buster:** Complex legal rights explained in simple language.
- **Voice Accessible:** Integrated Text-to-Speech for legal guidance.
- **Instant Drafting:** Generates professional PDF-ready documents locally in-browser.

## 🛠️ Tech Stack
- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **State Management:** React 'use client' Hooks

## 🔒 Privacy Commitment
AWAAZ does not use a database. All inputs are processed in the client's browser state and are wiped upon refreshing or starting over.
